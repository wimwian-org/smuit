Add `@smuit/list` (v1) — a Material Design 3–style list built from the design spec on smuit design tokens. Ships a composable `List.Root` / `List.Item` pair in **Baseline** and **Expressive** variants, one- and two-line rows, leading/trailing snippet slots, supporting text, and static / button / link interactivity with a hover state layer and visible focus, plus the Expressive position-based corner shape (first/middle/last/only), neutral/primary/secondary/tertiary tints, and automatic light/dark theming. Selection, three-line layout, roving keyboard navigation, reordering, subheaders, and the Expressive press shape-morph are deferred to a later release.
